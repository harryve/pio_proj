# Berging sensor

Draait op LilyGO TTGO T3 LoRa32 868MHz V1.6.1 ESP32.

Specificaties:

- Voedingsspanning: 3.3V DC of 5V DC
- GPIO Spanning: 3.3V*
- ESP32-PICO-D4 chip (240MHz dual core processor)
- SRAM: 520KB
- Flashgeheugen: 4MB
- Ingebouwde Wi-Fi
- Ingebouwde Bluetooth
- SX1276 LoRa chip
- LoRa frequentie: 868MHz
- Ingebouwd 0.96 inch (128x64) OLED display (SSD1306)
- USB naar serieel converter: CP2102 of CH9102F (drivers)
- Ingebouwd Li-ion/Li-Po batterij oplaadcircuit: TP4054 chip kan tot 500mA laden
- Ingebouwde micro SD kaart aansluiting
- Ingebouwde aan/uit knop
- Batterij connector: 2p Molex Picoblade
- Afmetingen PCB: 64.5 x 27.0 mm
